## Installation

set environmental variable
  export AWS_ACCESS_KEY_ID=''
  export AWS_SECRET_ACCESS_KEY=''
  export AWS_END_POINT=''
  export AWS_BKUP_DIR=''

Add this line to your application's Gemfile:


    gem 's3_sync'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install s3_sync
